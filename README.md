## 🖥️ Sobre o Projeto

Este repositório contém o slide deck interativo da Urbanis, estruturado com foco em design de alta fidelidade e estética premium inspirada nos eventos da Apple (Keynote). 

O projeto apresenta o modelo de negócios de consultoria, o ativo tecnológico **UrbanScore** (metodologia de análise geoespacial com dados fragmentados transformados em score de região) e a simulação de operação inicial da empresa.

## ⚡ Recursos do Slide Deck

* **Navegação Inteligente**: Use as setas do teclado (`←` e `→`) ou toque/deslize na tela (mobile).
* **Teclas de Atalho (Presenter Hotkeys)**:
  * `F` — Alternar tela cheia (Fullscreen).
  * `H` — Voltar para o Lobby inicial (Home).
  * `R` — Abrir modal para reiniciar a apresentação.
* **Modo Apresentador (Roteiro)**: Permite visualizar as notas e falas sugeridas de cada slide em tempo real.
* **Design Responsivo & Fluido**: Grid responsivo com tipografia ajustada via funções CSS `clamp()`.

## 🛠️ Tecnologias Utilizadas

* **HTML5** & **CSS3** (Variáveis nativas, animações 3D de parallax e profundidade de cena).
* **Vanilla JavaScript** (Controle de estados dos slides, atalhos do apresentador e sincronização do lobby).
